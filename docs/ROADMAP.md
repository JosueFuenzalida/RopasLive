# Roadmap

## Prioridad actual

La app debe vender rapido durante el live. Todo lo demas debe apoyar esa operacion sin meter friccion.

## MVP inmediato

- Autocomplete de clientes.
- Precios frecuentes por categoria.
- Reimpresion de etiquetas.
- Cierre de live con backup obligatorio.
- Restaurar backup desde JSON.
- Edicion rapida de pedido.
- Busqueda por codigo, cliente y estado.

## Backend y datos

- Mantener IndexedDB como fuente local rapida.
- Enviar eventos al backend cuando haya conexion.
- Guardar snapshots JSON por live.
- Preparar estructura para analisis posterior.

## Impresion

- Mantener impresion mock mientras se valida el flujo.
- Luego evaluar ESC/POS por Web Bluetooth o APK con Capacitor.

## Analisis

- Ventas por live.
- Ventas por categoria.
- Mejores clientes.
- Ticket promedio.
- Pendientes de pago.
- Tiempo desde venta hasta pago, empaque y despacho.
